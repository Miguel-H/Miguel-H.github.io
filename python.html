What I Learned About Python

Python is a widely used programming language for general purpose programming. It is a relatively simple language for beginner programmers to learn. While learning the language and researching how to use for my project, I learned that there are several of Python libraries and third-party modules,suchas Pyaudio or NumPy, that add more functionality to the the coding environment. Working with Python I learned that programming takes patience, experimentation, and iteration. My favorite part of working with Python is executing my own graphic user interface (GUI) formy project. It was fun researching how to configure my GUI in way that would allow the user to interact with the program I developed by naming files or setting parameters.
