What I Learned About HTML

Html is the language for coding and describing the structure of Web pages.You can create headings, paragraphs, lists, tables, etc. In addition you can add videos, photos, and links on the Web pages. Html utilizes many attributes, to define or determine the different elements that make up the Web page. For example, the p attribute is defined as the code used to place text in the style of a paragraph on the Web page.
