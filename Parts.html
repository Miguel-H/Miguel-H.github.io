Parts and Budget

 

Voice Changer Budget

 

 

 

 

 

 

 

 

 

 

All items found below can/should be purchased on Amazon. All prices listed are based on Amazon listings as of 10/20/16.

1.      OfficeTec Wired Keyboard and Mouse Combo (KB202) by OfficeTec

·        Needed for programming raspberry pi.

 

2.      Sandisk 16GB 16G Micro SDHC Class 4 TF Memory Card with Micro SD Card Reader - Bulk Packed by SanDisk

·       Memory storage for the operating system and the program itself, so that it can be transferred to the raspberry pi.

 

3.      USB Microphone,Fifine Plug &Play Home Studio USB Condenser Microphone for Skype, Recordings for YouTube, Google Voice Search, Games(Windows/Mac)-K668 by Fifine

·       In order to record voice phrases into the operating system and program.

 

4.      Raspberry Pi Micro USB Cable with ON/OFF Switch - Easy Start / Reboot !

·       To easily power the system on/off.

 

5.      Lumsing 2-Port USB Wall Charger Hub 20W Charging Station with Foldable Plug Power Adapter for iPhone 6S Plus Smartphones and Tablets(Grey) by Lumsing

·       In order to provide power.

 

6.      TFT Touch Screen, Kuman 3.5 inch 320480 Resolution TFT LCD Display with Protective Case + 3 x Heat sinks+ Touch Pen for Raspberry Pi 3 Model B, Pi 2 Model B & Pi Model B+ SC11 by Kuman

·       For the user interface.

 

7.      ELENKER Portable Mini Clip-On USB Powered Stereo Speaker Soundbar for Notebook Laptop PC Desktop Tablet by ELENKER

·       To output the resulting sound.

8.      Raspberry Pi 3 Model B Motherboard by Raspberry Pi

·       The required hardware component.

 

For testing purposes, all items will be required at the same time.
